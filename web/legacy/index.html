多 AI 模型放射科报告评估系统

以人工自由文本报告 + 影像为输入，自动生成多模型 AI 候选报告、做单报告与成对评估、 Top-N 排名、human-vs-AI 对比，并汇总到批量医生评估与科室级统计。核心是可复现的 Python library， CLI / API 为薄入口。

人工报告 + 影像→ 模态识别→ 多模型生成→ 单/成对评估→ Top-N 排名→ 批量 / 科室统计→ 统一 JSON
Overview · 概况

系统定位与一句话概况

面向放射科报告的「定量 + 定性」评估基线系统：用固定传统工具做基线，用多个 AI 模型生成的报告做主要参照， 把单个医生、整批医生、整个科室都能放在同一套指标体系里比较。

🧩

可复现核心库

12 个工具 · 2 个模块 · 3 个工作流，全部为标准 Python 包；CLI / API 仅作薄入口，便于平台化。

🏥

本地模型优先

自动发现旧 medHarness 已就位的 report-trained 模型；不兼容时按显式来源标记走云端 / 本地 VLM fallback。

📊

可审计输出

每例都落统一 JSON，并产出 CSV / Markdown 分析表；52 例样本运行通过 validation 闸门。

当前里程碑：在真实样本 上完成 52 例端到端运行，6 名放射科医生、 8 个生成模型、81 份 AI 候选报告， 全部 52/52 为真实 OCR、0 mock，最终 validation passed 。
